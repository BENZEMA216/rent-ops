# 模式：tracker — 房源跟踪管理

查看和管理房源跟踪表。

## 功能

### 查看当前状态

读取 `${CLAUDE_SKILL_DIR}/data/listings.md`，按用户需求展示：

**默认视图：** 按状态分组展示

```
📋 房源跟踪 — {N} 条记录

约看中 ({N})
  #003 南山-荣上居 1室1厅 4500元 ⭐4.2
  #007 福田-xxx花园 2室1厅 5200元 ⭐4.0

已评估 ({N})
  #001 南山-xxx公寓 1室1厅 3800元 ⭐3.9
  #005 宝安-xxx小区 1室1厅 3200元 ⭐3.5

已看 ({N})
  #002 南山-xxx雅苑 1室1厅 4000元 ⭐4.1 — 采光不错但噪音大

放弃 ({N})
  #004 福田-xxx大厦 1室0厅 3500元 ⭐2.8 — 隔断
```

### 更新状态

用户说：
- "把 #003 标为已看" → 更新状态为 `已看`
- "放弃 #005，太偏了" → 更新状态为 `放弃`，备注加上原因
- "#003 约了周六下午看" → 更新状态为 `约看中`，备注加上时间

### 筛选和排序

用户说：
- "只看评分 4 分以上的" → 筛选 评分 >= 4.0
- "按评分排序" → 降序排列
- "看南山区的" → 按小区名筛选

### 状态流转规则

`发现` → `已评估` → `约看中` → `已看` → `谈价中` → `已签` / `放弃`

- 状态只能前进（除了任意状态 → `放弃`）
- 同一 URL 不创建重复条目

### 统计信息

在底部展示：
```
统计：总 {N} 条 | 已评估 {N} | 约看中 {N} | 已看 {N} | 已签 {N} | 放弃 {N}
平均评分：{X.X} | 平均租金：{XXXX}元
```
